# AI Usage Report — Assignment 1

## AI Tools Used
The main AI tool used in this project was ChatGPT.

## How AI Was Used

### 1. Project Planning
AI was used to help organize the project structure, including:
- folder structure
- required files
- section planning for the portfolio website

### 2. HTML Structure
AI helped generate and refine the HTML layout for the website, including:
- top navigation
- profile section
- project cards
- contact form
- footer structure

### 3. CSS Styling
AI was used to improve the visual design and layout, especially for:
- responsive grid and card layout
- color palette suggestions
- spacing and alignment
- mobile responsiveness
- menu button behavior on smaller screens

### 4. JavaScript Features
AI helped generate and explain the logic for:
- dark/light theme toggle
- greeting message based on time of day
- form validation
- success and error feedback
- mobile navigation toggle

### 5. Debugging and Revision
AI was also used to identify and fix issues such as:
- unresolved anchors in HTML
- missing form feedback container
- back-to-top button behavior
- mobile navigation not showing on smaller screens
- code mismatches between HTML and JavaScript

## Benefits of Using AI
Using AI helped:
- speed up development
- improve code organization
- generate ideas for layout and UI
- understand JavaScript interactions more clearly
- debug issues faster

## Challenges and Limitations
Some AI-generated suggestions needed revision before they could be used. For example:
- some code needed to be adjusted to match the final HTML structure
- some selectors were not needed in the final CSS
- some paths and anchor targets had to be corrected manually
- the final version still required manual testing and cleanup

## Learning Outcomes
Through using AI, I learned:
- how to structure a portfolio website clearly
- how to build responsive layouts with CSS
- how to use JavaScript for DOM interaction and validation
- how to review AI-generated code critically instead of copying it directly

## Responsible Use and Modifications
I reviewed and edited all AI-generated suggestions before using them. I changed the wording, layout, code structure, and styling to match my own project requirements. I also tested the final result manually to ensure correctness, originality, and understanding. AI was used as a support tool for learning and improvement, not as a replacement for understanding the work.
